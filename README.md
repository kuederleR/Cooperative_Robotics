# Cooperative Robotics

This repository contains code and resources related to cooperative robotics, which involves multiple robots working together to achieve a common goal. The project focuses on developing algorithms and strategies for coordination, communication, and task allocation among robots in various scenarios.

## ROS Packages

- `aruco_pose_publisher`: A ROS 2 node that detects ArUco markers from a video stream and publishes their poses.
  - Read the full documentation for this package in [here](src/aruco_pose_publisher/).
